Порядок установки службы
1. В файле Install.bat в параметре binPath дописать путь к файлу PlanningService.exe, например, binPath = "C:\PlanningService\PlanningService.exe", остальные параметры оставить без изменений
2. Запустить файл Install.bat от имени администратора, убедиться что служба появилась в ПанельУправления - Администрирование - Службы, саму службу не запускать
3. Прописать настройки,необходимые для корректной работы службы в файл PlanningServiceConfig.xml (если найстройки указаны не будут - служба не запуститься)
Описание настроек:
	RootPath - путь к каталогу, где будет храниться лог обработки файлов
	InputFileDirPath - путь к каталогу, куда будут поступать инвойсы
	LogDirPath - путь к каталогу, в котором будут храниться обработанные инвойсы
	FileInvoiceCustomMask - маска для файлов инвойсов с таможни, например, InvoiceCustom
	FileInvoiceProductionMask - макска для файлов инвойсов от поставщика InvoiceTimeSlot
	TimerInterval - интервал просмотра каталога входящих файлов в секундах
	ServerName - имя сервера БД, где находится база Planning
	PlanningBaseName - имя БД planning
	PlanningBaseLogin - имя пользователя БД
	PlanningBasePwd - пароль 
	
4. Запустить службу, после запуска службы в папке с исполняемым файлов службы должен появиться файл PlanningServieLog.txt, в котором фиксируюся этапы запуска службы и ошибки
